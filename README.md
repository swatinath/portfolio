# portfolio
A personal portfolio website built using HTML, CSS and Bootstrap5

The project is divided into following sections :
1. Navbar (from Bootstrap)
2. Photograph and description
3. A list of Languages known to the developer
4. Projects (represented as cards from Bootstrap)
5. Skills of the developer (image along with specific details of domain knowledge; grid used)
6. Soft skills possessed by the developer
7. Cretifications (carousel from Bootstrap)
8. Education (list with logo of institute, name of the degree, marks/grades)
9. Footer (containing contact details and link to Github and LinkedIn profile of the developer)
